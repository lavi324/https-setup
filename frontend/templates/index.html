Welcome to the Football League Standings API. This service provides read-only access to football league tables (standings) via a simple HTTP API.

API Endpoint

The API has one GET endpoint to retrieve league standings:

  • /api/standings/<league_name> ‚Äì Returns the standings for the specified league in JSON format.

Replace <league_name> with the identifier of the league you want. For example:

  • /api/standings/premierleague ‚Äì Standings for Premier League.
  • /api/standings/laliga ‚Äì Standings for La Liga.
  • /api/standings/seriea ‚Äì Standings for Serie A.
  • api/standings/bundesliga - Standings for Bundesliga.
  • api/standings/ligue1 - Standings for Ligue 1.

Each object in the array represents a team‚Äôs standing, with fields like position, team name, number of matches played (played), wins, draws, losses, and total points. The data is read-only; updates must be made in the database directly.

Usage

To use the API, make GET requests to the endpoint (for example, using a web browser or curl). The landing page you are viewing is static documentation; the actual data comes from the API endpoint.
